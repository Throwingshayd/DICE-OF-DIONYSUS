Dice of Dionysus

You opened the repository root. There is no game entry here on purpose — the app lives in game/ and is served by Vite.

Recommended (one port: 3000):

  1. In the terminal: npm run dev
  2. Open http://localhost:3000/
  3. Or press F5 and choose Play game (Vite + Chrome).

If you use Go Live / Live Server: this repo sets liveServer.settings.root to game (see .vscode/settings.json). Live Server uses port 5501; Microsoft Live Preview uses 5500. Use 3000 for npm run dev (Vite) — do not run two servers on the same port.
