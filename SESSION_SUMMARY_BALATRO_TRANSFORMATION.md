## 🎓 What Was Learned

### Design Principles Applied
1. **Anticipation → Payoff** - Build-up before big moments
2. **Stagger Everything** - Sequential > simultaneous
3. **Color Code Information** - Blue pips, red favour, gold money
4. **Particle Effects** - Celebrate achievements
5. **Screen Shake** - Physical impact for big scores
6. **Smooth Transitions** - No jarring jumps
7. **Visual Hierarchy** - Important things glow/pulse
8. **Tactile Feedback** - Buttons respond to clicks

### Technical Patterns Established
- **Callback-based animations** for sequencing
- **CSS custom properties** for dynamic values
- **requestAnimationFrame** for smooth counting
- **Auto-cleanup** for particles (no memory leaks)
- **Defensive programming** with fallbacks
- **Logger system** for clean debugging
